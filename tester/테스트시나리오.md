> Jest 를 활용하여 테스트의 구조를 정의해본다.   
공통으로 해야 부분과는 분리되어야 한다.   
줄바꿈으로 다른 내용을 표현한다.

+ 기본 검사
    - `do_clear()`
        - 파일 유무 검사 : **없는지**
        - 초기화 검사
    - `do_publish()`
        - 파일 유무 검사 
        - 파일 내용 검사
    - `이름 변경 로딩 검사`
    - 파일내용 검사 : 스냅샷
    - 속성 검사

> 하나의 test 는 하나의 test 폴더를 기준으로 한다.
