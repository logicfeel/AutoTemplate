# 테스트 시나리오

오토템플릿으로 만들수 있는 시나리오를 작성해 보자.

* 내용
    - 하나를 제작해 두면, 두고두고 사용한다.
    - scope
        - 파일의 구조를 part 공통화해서 사용하는 방법
        - 지역 partf를 통한
    - extends
        - 사용할 모듈을 확장할 때 경우
    - cover
        - 사용할 모듈을 일부 재정의 할 경우
    - import 
        - 필요한 모듈을 묶어서 새로 정의할 경우
    - group
        - 생성파일의 묶음(src의 묶음 역활 => page)
            - 인스턴스화 가능
